Скрипт для объединения нескольких баз данных от домофонов Метаком MK2012 и MK2003.2, в одну базу данных контроллера ELC-T4E-5000.

Формат базы данны от блоков вызова Метаком MK2012 и MK2003.2 не совпадает с форматом контроллера ELC-T4E-5000, из-за этого перенести базу с помощью ключа Dallas iButtontm DS1996 невозможно. Метакомовская программа MKA-02U конвертирует какую-то лютую хрень котороую невозможно залить на контроллер никак (может, конечно и можно но у меня не получилось, легче было скрипт написать)

Как работать со скриптом:
1. Через Метакомовскую программу MKA-02U снять информацию с микросхем памяти блоков вызова (МК2012 или МК2003.2).
Настройки программы при чтении микросхемы:
	Модель домофона: Неизвестный домофон
	Тип носителя: 24CXX
	Объём памяти: С256
	Сохранить с расширением файла .mkf
2. Сохранить файлы .mkf с блоков вызова в папку 2012
3. Запустить скрипт
4. В результате работы скрипта создастся файл (в корневой папке скрипта) elc-new.mkf
5. Открыть и записать на микросхему ELC-T4E-5000, через ту же Метакомовскую программу MKA-02U, с настройками как и в пункте 1
